\doxysubsection{Detailed Description}
Represents a wizard that processes rule files and manages file data collection. 

The \doxylink{class_file_sorting_wiz_1_1_wizard}{Wizard} class provides functionality to initialize components and collect file data based on a specified rule file and directory paths. It is designed to handle operations related to rule file deserialization and logging.
